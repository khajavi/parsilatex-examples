اگر در یک سند، تعداد شکل‌ها زیاد باشد، شاید دوست داشته باشید که علاوه بر عنوان شکل‌ها در فهرست شکل‌ها، نسخه کوچک شده (thumbnail) شکل‌ها را هم در ابتدای عنوان شکل‌ها نمایش دهید.
